POLÍTICA DE ACCESIBILIDAD

En la construcción del sitio web de "LOTO IA.COM" se han adoptado una serie de medidas con el objetivo de mejorar su accesibilidad. Esto conlleva una serie de ventajas como:

  • Facilitar el acceso de los usuarios independientemente de su condición física o de su entorno
  • Permitir el acceso con diferentes agentes de usuario
  • Incluir contenidos claros bien estructurados
  • Mejorar la navegación y la experiencia del usuario

Entre otras se han adoptado las siguientes medidas:

  • Uso de CSS para la presentación de la información
  • Texto alternativo en las imágenes
  • Los enlaces ofrecen detalles de la función o destino del hipervínculo
  • Uso de los estándares del W3C
  • Periódicamente se realizan mejoras en la web que siempre estarán orientadas a mantener o elevar el nivel de cumplimiento de estándares

Cumplimiento de los estándares

Las páginas de este sitio web se adaptan a las "Pautas de Contenido Accesible en web 1.0" del W3C, disponible en http://www.w3.org/TR/1999/WAI-WEBCONTENT-19990505, nivel A (A).

Todos los requisitos de Prioridad 1 han sido comprobados con un análisis manual de la accesibilidad a través de diferentes herramientas semi-automáticas, agentes de usuario y ayudas técnicas.

Configuración recomendada para navegar por la web.

Para la mejor experiencia en el uso de este canal de relación, se recomienda al Participante el empleo de los siguientes medios técnicos:

  • Navegador: Internet Explorer 7 o superior, Mozilla 3.0.0 o superior.
  • Resolución de pantalla recomendada: 1.024x768
  • Para ver algunos de los contenidos, será necesario el plugin Adobe Flash
  • Para consultar algunos de los documentos será necesario tener instalado el plugin Adobe Reader
  • Debes tener activadas en tu navegador las opciones de Java y JavaScript
  • Debes tener habilitadas las cookies en tu navegador

No existen requisitos ni restricciones respecto al ordenador o sistema operativos empleados.
